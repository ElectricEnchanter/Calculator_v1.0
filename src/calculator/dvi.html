Я тут склепал чота вроде калькулятора

Чтобы установить это чудо заходи в src:

cd src

И запускай через мейк:

make run

Если захочешь удОлить Смарт КАЛк жмякай:

make uninstall

Такж можно заархивировать исполнительные файлы:

make dist

Полезные функции этой программы

•   Можно посчитать практически любые математические выражения с подстановкой переменной x в виде числа.

•   Можно построить практически любые графики с подстановкой переменной x в виде числа.

Чуть позже будут доступны новые функции:

•   Кредитный калькулятор.

•   Депозитный калькулятор.
